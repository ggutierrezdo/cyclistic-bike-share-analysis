# cyclistic-bike-share-analysis
Data analysis of Cyclistic bike-share usage comparing casual riders vs members using R and Tableau.


## Project Overview
This project analyzes bike-share usage data from Cyclistic to identify behavioral differences between casual riders and annual members.

The objective is to support business decisions aimed at converting casual riders into annual members.

## Tools Used
- R (tidyverse, lubridate, janitor)
- Tableau Public

## Data Source
Public bike-share trip data provided by Cyclistic (12 months).

## Key Steps
- Data cleaning and transformation in R
- Feature engineering (ride duration, weekday, hour, month)
- Outlier removal using percentile method
- Aggregation for visualization
- Interactive dashboard creation in Tableau

## Key Insights
- Casual riders take fewer trips but have longer average ride durations
- Members show more consistent usage across weekdays
- Peak usage times differ between casual riders and members
- Station usage patterns suggest different trip purposes
